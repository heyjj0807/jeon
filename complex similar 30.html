복소해석학 기출 기반 유사 문제 30선

본 자료는 복소해석학의 핵심 개념인 해석함수, 복소적분, 급수, 유수 이론 등을 종합적으로 점검할 수 있도록 구성되었습니다.
각 문제의 '정답 및 해설 보기' 버튼을 클릭하여 학습을 진행하세요.

문항 1: 복소수의 극형식

방정식 \( z^4 = -8 - 8\sqrt{3}i \) 를 만족하는 모든 복소수 \(z\)를 구하시오.

정답 및 해설

먼저 우변의 복소수를 극형식으로 나타낸다. \(w = -8 - 8\sqrt{3}i\).
\(|w| = \sqrt{(-8)^2 + (-8\sqrt{3})^2} = \sqrt{64 + 192} = \sqrt{256} = 16\).
\(w = 16(-\frac{1}{2} - \frac{\sqrt{3}}{2}i) = 16(\cos\frac{4\pi}{3} + i\sin\frac{4\pi}{3})\).
\(z = r(\cos\theta + i\sin\theta)\)라 하면, 드무아브르 정리에 의해 \(z^4 = r^4(\cos(4\theta) + i\sin(4\theta))\)이다.
따라서 \(r^4 = 16 \implies r=2\) 이고, \(4\theta = \frac{4\pi}{3} + 2k\pi \implies \theta = \frac{\pi}{3} + \frac{k\pi}{2}\) (k=0, 1, 2, 3).
k=0: \(z_0 = 2(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}) = 1 + \sqrt{3}i\)
k=1: \(z_1 = 2(\cos\frac{5\pi}{6} + i\sin\frac{5\pi}{6}) = -\sqrt{3} + i\)
k=2: \(z_2 = 2(\cos\frac{4\pi}{3} + i\sin\frac{4\pi}{3}) = -1 - \sqrt{3}i\)
k=3: \(z_3 = 2(\cos\frac{11\pi}{6} + i\sin\frac{11\pi}{6}) = \sqrt{3} - i\)
정답: \(1+\sqrt{3}i, -\sqrt{3}+i, -1-\sqrt{3}i, \sqrt{3}-i\)

문항 2: 코시-리만 방정식

함수 \(f(z) = e^x(\cos y + i \sin y)\)가 해석적인지 판별하고, 만약 해석적이라면 그 도함수 \(f'(z)\)를 구하시오. (단, \(z=x+iy\))

정답 및 해설

\(f(z) = u(x,y) + iv(x,y)\)에서 \(u = e^x \cos y\), \(v = e^x \sin y\)이다.
편도함수를 구하면:
\(u_x = e^x \cos y\), \(u_y = -e^x \sin y\)
\(v_x = e^x \sin y\), \(v_y = e^x \cos y\)
코시-리만 방정식 확인:
\(u_x = v_y = e^x \cos y\) (성립)
\(u_y = -v_x = -e^x \sin y\) (성립)
모든 편도함수는 연속이므로 \(f(z)\)는 모든 복소평면에서 해석적(entire)이다.
도함수는 \(f'(z) = u_x + iv_x = e^x \cos y + i(e^x \sin y) = e^x(\cos y + i\sin y) = e^z\).
결론: 해석적이며, \(f'(z) = e^z\) 이다.

문항 3: 조화함수와 조화켤레함수

함수 \(u(x,y) = x^3 - 3xy^2 + y\)가 조화함수임을 보이고, \(u\)의 조화켤레함수 \(v(x,y)\)를 구하시오. (단, \(v(0,0)=0\))

정답 및 해설

1. 조화함수 증명: 라플라스 방정식 \(u_{xx} + u_{yy} = 0\)을 만족하는지 확인한다.
\(u_x = 3x^2 - 3y^2\), \(u_{xx} = 6x\)
\(u_y = -6xy + 1\), \(u_{yy} = -6x\)
\(u_{xx} + u_{yy} = 6x - 6x = 0\). 따라서 \(u\)는 조화함수이다.
2. 조화켤레함수 구하기: 코시-리만 방정식을 이용한다.
\(v_y = u_x = 3x^2 - 3y^2\). y에 대해 적분하면 \(v(x,y) = 3x^2y - y^3 + h(x)\).
\(v_x = 6xy + h'(x)\).
\(v_x = -u_y = -(-6xy + 1) = 6xy - 1\).
따라서 \(6xy + h'(x) = 6xy - 1 \implies h'(x) = -1 \implies h(x) = -x + C\).
\(v(x,y) = 3x^2y - y^3 - x + C\).
조건 \(v(0,0)=0\)을 대입하면 \(0 = 0 - 0 - 0 + C \implies C=0\).
조화켤레함수: \(v(x,y) = 3x^2y - y^3 - x\)

문항 4: 복소 선적분

경로 \(C\)가 \(z(t) = 2e^{it}\) (\(0 \le t \le \pi\))일 때, 선적분 \( \int_C \bar{z} \,dz \)의 값을 구하시오.

정답 및 해설

주어진 경로 \(C\)는 중심이 원점이고 반지름이 2인 원의 상반부이다.
\(z(t) = 2e^{it} = 2(\cos t + i\sin t)\)
\(\bar{z}(t) = 2e^{-it} = 2(\cos t - i\sin t)\)
\(dz = z'(t)dt = 2ie^{it} dt\)
적분을 t에 대한 정적분으로 변환한다.
\( \int_C \bar{z} \,dz = \int_0^\pi (2e^{-it})(2ie^{it}) \,dt \)
\( = \int_0^\pi 4i \cdot e^0 \,dt = \int_0^\pi 4i \,dt \)
\( = [4it]_0^\pi = 4i\pi - 0 \)
정답: \(4i\pi\)

문항 5: 코시의 적분 정리

경로 \(C\)가 정사각형 \(|x|\le 2, |y|\le 2\)일 때, 적분 \( \oint_C \frac{z^2}{z^2+9} \,dz \)의 값을 구하시오.

정답 및 해설

피적분함수 \(f(z) = \frac{z^2}{z^2+9}\)의 특이점은 \(z^2+9=0\), 즉 \(z=\pm 3i\)이다.
적분 경로 C는 \(x=\pm 2, y=\pm 2\)로 둘러싸인 정사각형 영역이다.
특이점 \(z=3i\)와 \(z=-3i\)는 모두 이 정사각형 경로 외부에 존재한다. (\(|Im(z)|=3 > 2\))
피적분함수는 경로 C와 그 내부에서 해석적이므로, 코시의 적분 정리(Cauchy's Integral Theorem)에 의해 적분 값은 0이다.
정답: 0

문항 6: 코시의 적분 공식

경로 \(C\)가 원 \(|z-1|=2\)일 때, 적분 \( \oint_C \frac{e^{2z}}{z-2} \,dz \)의 값을 구하시오. (경로는 양의 방향)

정답 및 해설

피적분함수의 특이점은 \(z=2\)이다.
적분 경로 C는 중심이 1이고 반지름이 2인 원이다. 점 \(z=2\)는 \(|2-1|=1 < 2\)이므로 경로 내부에 존재한다.
함수 \(f(z) = e^{2z}\)는 모든 복소평면에서 해석적이다.
코시의 적분 공식 \(\oint_C \frac{f(z)}{z-z_0} dz = 2\pi i f(z_0)\)을 이용한다.
여기서 \(z_0=2\), \(f(z)=e^{2z}\)이다.
적분 값 = \(2\pi i \cdot f(2) = 2\pi i \cdot e^{2(2)} = 2\pi i e^4\).
정답: \(2\pi i e^4\)

문항 7: 코시의 도함수 공식

경로 C가 원 \( |z|=2 \) (양의 방향)일 때, 다음 적분 값을 구하시오. $$ \oint_C \frac{\cos(z)}{(z-\frac{\pi}{2})^3} \,dz $$

정답 및 해설

특이점 \(z_0 = \frac{\pi}{2}\)는 경로 내부에 존재한다. (\(|\frac{\pi}{2}| \approx 1.57 < 2\))
코시의 도함수 공식 \( \oint_C \frac{f(z)}{(z-z_0)^{n+1}} \,dz = \frac{2\pi i}{n!} f^{(n)}(z_0) \)을 이용한다.
여기서 \(f(z) = \cos(z)\), \(z_0 = \frac{\pi}{2}\), \(n+1=3 \implies n=2\)이다.
\(f'(z) = -\sin(z)\)
\(f''(z) = -\cos(z)\)
따라서, \(f''(\frac{\pi}{2}) = -\cos(\frac{\pi}{2}) = 0\).
적분 값은 \(\frac{2\pi i}{2!} f''(\frac{\pi}{2}) = \pi i \cdot 0 = 0\).
정답: 0

문항 8: 리우빌의 정리

함수 \(f(z)\)는 모든 복소평면에서 해석적(entire)이고, 모든 \(z\)에 대하여 \(|f(z)| \le \sqrt{|z|} + 10\)을 만족한다. \(f(z)\)가 상수함수임을 보이시오.

정답 및 해설

임의의 점 \(z_0\)에 대해 \(f'(z_0)=0\)임을 보인다. 코시의 부등식 \( |f'(z_0)| \le \frac{M_R}{R} \)를 이용한다.
중심이 \(z_0\)이고 반지름이 R인 원 \(C_R: |z-z_0|=R\) 위에서 \(|z| \le |z_0|+R\)이다.
따라서 \(|f(z)| \le \sqrt{|z|} + 10 \le \sqrt{|z_0|+R} + 10\) 이므로 \(M_R = \sqrt{|z_0|+R} + 10\)으로 잡을 수 있다.
\(|f'(z_0)| \le \frac{\sqrt{|z_0|+R} + 10}{R}\).
여기서 \(R \to \infty\)의 극한을 취하면, \( \lim_{R \to \infty} \frac{\sqrt{|z_0|+R} + 10}{R} = 0 \) 이다.
따라서 임의의 \(z_0\)에 대해 \(|f'(z_0)| \le 0\) 이므로 \(f'(z_0)=0\)이다.
모든 점에서 도함수가 0이므로 \(f(z)\)는 상수함수이다. (이는 리우빌 정리의 일반화된 형태이다.)

문항 9: 테일러 급수

함수 \(f(z) = \frac{1}{1+z^2}\)을 \(z=0\)을 중심으로 테일러 급수로 전개하고, 그 수렴반경을 구하시오.

정답 및 해설

함수 \(f(z)\)는 등비급수 \(\frac{1}{1-w} = \sum_{n=0}^{\infty} w^n\) (단, \(|w|<1\))를 이용하여 전개할 수 있다.
\(w = -z^2\)로 치환하면, \(f(z) = \frac{1}{1-(-z^2)} = \sum_{n=0}^{\infty} (-z^2)^n = \sum_{n=0}^{\infty} (-1)^n z^{2n}\).
이 급수는 \(|w| < 1\), 즉 \(|-z^2| < 1 \implies |z|^2 < 1 \implies |z|<1\)일 때 수렴한다.
테일러 급수: \(\sum_{n=0}^{\infty} (-1)^n z^{2n} = 1 - z^2 + z^4 - z^6 + \dots\), 수렴반경: 1
(참고: 수렴반경은 중심 \(z=0\)에서 가장 가까운 특이점(\(z=\pm i\))까지의 거리와 같다.)

문항 10: 로랑 급수

함수 \(f(z) = \frac{1}{(z-1)(z-3)}\)을 영역 \(1 < |z| < 3\)에서 로랑 급수로 전개하시오.

정답 및 해설

먼저 부분분수로 분해한다: \(f(z) = \frac{1}{2}\left(\frac{1}{z-3} - \frac{1}{z-1}\right)\).
영역 \(1 < |z| < 3\)에 맞게 각 항을 등비급수 형태로 변형한다.
1) \(|\frac{z}{3}| < 1\)이므로, \(\frac{1}{z-3} = \frac{-1}{3(1-z/3)} = -\frac{1}{3}\sum_{n=0}^{\infty} (\frac{z}{3})^n = -\sum_{n=0}^{\infty} \frac{z^n}{3^{n+1}}\).
2) \(|\frac{1}{z}| < 1\)이므로, \(\frac{-1}{z-1} = \frac{-1}{z(1-1/z)} = -\frac{1}{z}\sum_{n=0}^{\infty} (\frac{1}{z})^n = -\sum_{n=0}^{\infty} \frac{1}{z^{n+1}}\).
따라서 \(f(z) = \frac{1}{2} \left( -\sum_{n=0}^{\infty} \frac{z^n}{3^{n+1}} - \sum_{n=0}^{\infty} \frac{1}{z^{n+1}} \right)\).
로랑 급수: \(-\frac{1}{2} \sum_{n=0}^{\infty} \frac{z^n}{3^{n+1}} - \frac{1}{2} \sum_{n=1}^{\infty} \frac{1}{z^n}\)

문항 11: 유수(Residue) 계산

함수 \(f(z) = \frac{z^2}{(z-1)^2(z+1)}\)의 모든 특이점에서 유수를 구하시오.

정답 및 해설

특이점은 \(z=1\) (위수 2인 극)과 \(z=-1\) (단순극)이다.
1) \(z=-1\)에서의 유수 (단순극 공식):
\(Res(f, -1) = \lim_{z \to -1} (z+1)f(z) = \lim_{z \to -1} \frac{z^2}{(z-1)^2} = \frac{(-1)^2}{(-1-1)^2} = \frac{1}{4}\).
2) \(z=1\)에서의 유수 (위수 m인 극 공식):
\(Res(f, 1) = \frac{1}{(2-1)!} \lim_{z \to 1} \frac{d}{dz} \left[ (z-1)^2 f(z) \right] = \lim_{z \to 1} \frac{d}{dz} \left[ \frac{z^2}{z+1} \right]\).
\(\frac{d}{dz} \left[ \frac{z^2}{z+1} \right] = \frac{2z(z+1)-z^2(1)}{(z+1)^2} = \frac{z^2+2z}{(z+1)^2}\).
극한값은 \( \frac{1^2+2(1)}{(1+1)^2} = \frac{3}{4} \).
\(Res(f, -1) = \frac{1}{4}\), \(Res(f, 1) = \frac{3}{4}\)

문항 12: 유수 정리

경로 C가 원 \( |z|=3 \) (양의 방향)일 때, 유수 정리를 이용하여 다음 적분 값을 구하시오. $$ \oint_C \frac{e^z}{(z-1)(z+2)^2} \,dz $$

정답 및 해설

특이점 \(z=1\) (단순극)과 \(z=-2\) (위수 2인 극)는 모두 경로 \(|z|=3\) 내부에 존재한다.
유수 정리에 의해 적분 값은 \(2\pi i \sum Res\)이다.
1) \(Res(f, 1) = \lim_{z \to 1} (z-1)f(z) = \lim_{z \to 1} \frac{e^z}{(z+2)^2} = \frac{e^1}{(1+2)^2} = \frac{e}{9}\).
2) \(Res(f, -2) = \lim_{z \to -2} \frac{d}{dz} \left[ (z+2)^2 f(z) \right] = \lim_{z \to -2} \frac{d}{dz} \left[ \frac{e^z}{z-1} \right]\).
\(\frac{d}{dz} \left[ \frac{e^z}{z-1} \right] = \frac{e^z(z-1)-e^z}{(z-1)^2} = \frac{e^z(z-2)}{(z-1)^2}\).
극한값은 \( \frac{e^{-2}(-2-2)}{(-2-1)^2} = \frac{-4e^{-2}}{9}\).
적분 값 = \(2\pi i \left( \frac{e}{9} - \frac{4e^{-2}}{9} \right)\).
정답: \( \frac{2\pi i}{9} (e - 4e^{-2}) \)

문항 13: 실수 이상적분 계산

유수 정리를 이용하여 다음 실수 이상적분의 값을 구하시오. $$ \int_{-\infty}^{\infty} \frac{1}{x^2+2x+2} \,dx $$

정답 및 해설

복소함수 \(f(z) = \frac{1}{z^2+2z+2}\)를 생각한다. 특이점은 \(z^2+2z+2=0 \implies z = \frac{-2 \pm \sqrt{4-8}}{2} = -1 \pm i\).
상반평면에 있는 특이점은 \(z_0 = -1+i\)이다. (단순극)
경로를 반지름 R인 상반원 \(C_R\)과 실수축 구간 \([-R, R]\)으로 잡으면, R이 충분히 클 때 경로 내부에 \(z_0\)가 포함된다.
\( \int_{-R}^{R} f(x)dx + \int_{C_R} f(z)dz = 2\pi i \cdot Res(f, -1+i)\).
\(Res(f, -1+i) = \lim_{z \to -1+i} (z-(-1+i))f(z) = \lim_{z \to -1+i} \frac{1}{z-(-1-i)} = \frac{1}{(-1+i)-(-1-i)} = \frac{1}{2i}\).
\(R \to \infty\)일 때, \(|z^2 f(z)| \to 1\)이므로 조르단의 보조정리에 의해 \(\int_{C_R} f(z)dz \to 0\).
따라서 \( \int_{-\infty}^{\infty} f(x)dx = 2\pi i \cdot \frac{1}{2i} = \pi\).
정답: \(\pi\)

문항 14: 최대 절댓값 원리

함수 \(f(z)=z^2+2i\)가 닫힌 원판 \(D = \{z \in \mathbb{C} : |z| \le 1\}\)에서 가질 수 있는 \(|f(z)|\)의 최댓값을 구하시오.

정답 및 해설

최대 절댓값 원리(Maximum Modulus Principle)에 의해, 상수함수가 아닌 해석함수의 절댓값은 영역의 내부에서 최댓값을 갖지 않고, 경계에서 최댓값을 갖는다.
따라서 우리는 경계 \(|z|=1\)에서 최댓값을 찾으면 된다. \(z=e^{i\theta}\)로 놓자.
\(f(z) = (e^{i\theta})^2 + 2i = e^{2i\theta} + 2i = (\cos(2\theta) + i\sin(2\theta)) + 2i = \cos(2\theta) + i(\sin(2\theta)+2)\).
\(|f(z)|^2 = (\cos(2\theta))^2 + (\sin(2\theta)+2)^2\)
\( = \cos^2(2\theta) + \sin^2(2\theta) + 4\sin(2\theta) + 4\)
\( = 1 + 4\sin(2\theta) + 4 = 5 + 4\sin(2\theta)\).
\(|f(z)|^2\)는 \(\sin(2\theta)=1\)일 때 최댓값을 갖는다. 최댓값은 \(5+4(1)=9\).
따라서 \(|f(z)|\)의 최댓값은 \(\sqrt{9}=3\).
정답: 3

문항 15: 특이점의 분류

함수 \( f(z) = \frac{1-\cos z}{z^2} \) 의 특이점 \(z=0\)의 종류를 판별하시오. (제거 가능 특이점, 극, 본질적 특이점)

정답 및 해설

특이점의 종류를 판별하는 방법은 극한을 이용하거나 로랑 급수를 이용하는 것이다.
1. 극한 이용: \( \lim_{z \to 0} f(z) \)의 존재 여부를 확인한다.
\(\cos z\)의 테일러 전개: \(\cos z = 1 - \frac{z^2}{2!} + \frac{z^4}{4!} - \dots\)
\( \lim_{z \to 0} \frac{1-(1 - \frac{z^2}{2!} + \frac{z^4}{4!} - \dots)}{z^2} = \lim_{z \to 0} \frac{\frac{z^2}{2} - \frac{z^4}{24} + \dots}{z^2} = \lim_{z \to 0} (\frac{1}{2} - \frac{z^2}{24} + \dots) = \frac{1}{2}\).
극한값이 유한하게 존재하므로, \(z=0\)은 제거 가능 특이점(removable singularity)이다.
정답: 제거 가능 특이점

문항 16: 삼각함수 실수적분

유수 정리를 이용하여 다음 정적분의 값을 구하시오. $$ \int_{0}^{2\pi} \frac{1}{5 - 4\cos\theta} \,d\theta $$

정답 및 해설

\(z=e^{i\theta}\)로 치환하면, \(d\theta = \frac{dz}{iz}\)이고 \(\cos\theta = \frac{1}{2}(z+\frac{1}{z})\)이다. 적분 경로는 단위 원 C가 된다.
적분식은 다음과 같이 변환된다.
\( \oint_C \frac{1}{5 - 4 \cdot \frac{1}{2}(z+z^{-1})} \cdot \frac{dz}{iz} = \oint_C \frac{1}{5 - 2(z+z^{-1})} \cdot \frac{dz}{iz} \)
\( = \oint_C \frac{1}{-2iz + 5i - 2iz^{-1}} dz = \oint_C \frac{z}{-2iz^2 + 5iz - 2i} dz \)
\( = \frac{1}{i} \oint_C \frac{-z}{2z^2 - 5z + 2} dz = \frac{1}{i} \oint_C \frac{-z}{(2z-1)(z-2)} dz \).
단위 원 내부에 있는 특이점은 \(z=1/2\) 뿐이다.
피적분함수 \(f(z) = \frac{-z}{(2z-1)(z-2)}\)의 \(z=1/2\)에서의 유수는
\(Res(f, 1/2) = \lim_{z \to 1/2} (z-1/2) \frac{-z}{2(z-1/2)(z-2)} = \frac{-(1/2)}{2(1/2-2)} = \frac{-1/2}{2(-3/2)} = \frac{-1/2}{-3} = \frac{1}{6}\).
따라서 적분 값은 \(\frac{1}{i} \cdot (2\pi i \cdot Res) = \frac{1}{i} \cdot 2\pi i \cdot \frac{1}{6} = \frac{2\pi}{6} = \frac{\pi}{3}\).
정답: \(\frac{\pi}{3}\)

문항 17: 인수의 원리

함수 \(f(z) = z^4 - 2z^3 + z^2 - 1\)이 원 \(|z|=1\)의 내부에 몇 개의 근(zero)을 갖는지 구하시오.

정답 및 해설

루셰의 정리(Rouché's Theorem)를 이용한다. \(f(z) = g(z) + h(z)\)로 분해하여 경계 \(|z|=1\)에서 \(|g(z)| > |h(z)|\)임을 보인다.
\(g(z) = -2z^3\)으로 두고, \(h(z) = z^4 + z^2 - 1\)로 두자.
경계 \(|z|=1\)에서:
\(|g(z)| = |-2z^3| = 2|z|^3 = 2(1)^3 = 2\).
\(|h(z)| = |z^4 + z^2 - 1| \le |z^4| + |z^2| + |-1| = |z|^4 + |z|^2 + 1 = 1+1+1=3\).
이 경우 \(|g(z)| > |h(z)|\)가 성립하지 않는다. 다른 방법으로 시도한다.

\(g(z) = z^4 - 2z^3\)으로 두고, \(h(z) = z^2 - 1\)로 두자.
경계 \(|z|=1\)에서:
\(|g(z)| = |z^3(z-2)| = |z|^3 |z-2| = 1 \cdot |z-2|\). \(z=e^{i\theta}\)일 때, \(z-2\)는 중심이 -2이고 반지름이 1인 원 위의 점이다. 따라서 \(1 \le |z-2| \le 3\).
\(|h(z)| = |z^2 - 1| \le |z^2| + |-1| = 2\).
이 또한 명확하지 않다.

다시, \(g(z) = -2z^3\), \(h(z) = z^4 + z^2 - 1\)로 돌아가자. 위에서 \(|h(z)|\le 3\)으로 추정했지만 더 정확하게 보자.
\(|h(z)| \le |z|^4 + |z^2-1|\). \(|z^2-1|\)의 최댓값을 찾아야 한다. 이 방법은 복잡하다.

**가장 좋은 선택:** \(g(z) = -2z^3 + z^2\), \(h(z) = z^4 - 1\).
\(|g(z)| = |z^2(-2z+1)| = |-2z+1|\). \(z=e^{i\theta}\)일 때, \(-2z\)는 반지름 2인 원을 돌고, \(-2z+1\)은 중심 1, 반지름 2인 원을 돈다. \(1 \le |-2z+1| \le 3\).
\(|h(z)| = |z^4-1|\). \(|h(z)|^2 = (z^4-1)(\bar{z}^4-1) = z^4\bar{z}^4 - z^4 - \bar{z}^4 + 1 = 2 - 2\cos(4\theta)\). 최댓값은 4. \(|h(z)|\)의 최댓값은 2.
이것도 안된다.

**올바른 접근:** \(g(z) = -2z^3\), \(h(z) = z^4+z^2-1\).
\(|g(z)|=2\).
\(|h(z)| = |z^4+z^2-1| \leq |z|^4+|z|^2+|-1| = 1+1+1=3\). 이 부등식은 너무 약하다.
대신, \(z=e^{i\theta}\)일 때, \(h(z)=e^{i4\theta}+e^{i2\theta}-1\). \(|h(z)|\)가 2보다 작은지 확인하자.
예를 들어 \(z=i\)일 때 \(h(i) = i^4+i^2-1 = 1-1-1 = -1\), \(|h(i)|=1 < 2\).
예를 들어 \(z=-1\)일 때 \(h(-1)=1+1-1=1\), \(|h(-1)|=1 < 2\).
\(|h(z)| \le |z^4+z^2| + 1 = |z^2(z^2+1)|+1 = |z^2+1|+1 = |e^{i2\theta}+1|+1\). \(|e^{i2\theta}+1|\)의 최댓값은 \(2\theta=\pi\), 즉 \(z=i\)일 때 0이고, \(2\theta=0\), 즉 \(z=1\)일 때 2이다. 따라서 \(|h(z)|\)는 3까지 커질 수 있다. 이 선택은 실패다.

**가장 간단한 접근:** \(g(z) = z^4-2z^3\), \(h(z)=z^2-1\). 위에서 실패했다.
**최종 시도:** \(g(z) = -2z^3+z^2-1\), \(h(z) = z^4\).
\(|h(z)|=1\).
\(|g(z)| = |-2z^3+z^2-1| \ge |-2z^3| - |z^2-1| = 2 - |z^2-1|\).
\(|z^2-1|\)의 최댓값은 \(z=i\)일 때 \(|-1-1|=2\). 최솟값은 \(z=1\)일 때 0.
이 부등식은 \(|g(z)| \ge 0\)만을 보장한다.

**교재에 흔히 나오는 풀이:** \(g(z) = z^4 - 2z^3\), \(h(z)=z^2-1\)에서 \(|g(z)|=|z-2|\)는 [1,3]에 있고, \(|h(z)|=|z^2-1|\)는 [0,2]에 있다. 이것도 안된다.

**다시 처음으로:** \(g(z) = -2z^3\), \(h(z) = z^4+z^2-1\). \(|g(z)|=2\). \(|h(z)| \leq |z^4|+|z^2|+|-1| = 3\). 이 방법은 안된다.

**정답 접근법:** \(g(z) = z^2-2z^3\), \(h(z)=z^4-1\). 이것도 안된다.
아마도 문제는 \(f(z) = 2z^4 - 2z^3 + z^2 - 1\)이거나 해야 루셰 정리가 쉽게 풀린다. 주어진 문제 그대로는 쉽지 않다. 만약 문제가 \(f(z)=z^4-5z+1\) 이라면, \(g(z)=-5z\), \(h(z)=z^4+1\)로 두면 \(|g(z)|=5\), \(|h(z)|\le|z|^4+1=2\). \(|g|>|h|\)이므로 \(f(z)\)는 \(g(z)\)와 같은 수의 근, 즉 1개의 근을 갖는다.
**주어진 문제에 대한 해설:** (인수정리/루셰정리가 깔끔하게 적용되지 않는 예시일 수 있음. 출제오류 가능성) 만약 \(f(z) = z^3-2z+5\)이라면 \(g(z)=z^3\), \(h(z)=-2z+5\)일 때, \(|z|=2\)에서 \(|g|=8\), \(|h|\le 2|z|+5=9\) 안됨. \(g(z)=5, h(z)=z^3-2z\). \(|g|=5\), \(|h|\le |z|^3+2|z|=8+4=12\) 안됨.

**이 문제는 인수의 원리 또는 루셰의 정리의 전형적인 예제가 아니므로, 다른 문제로 대체합니다.**

**대체 문항 17:** 함수 \(f(z) = z^5 + 3z^2 + 1\)이 영역 \(1 < |z| < 2\)에 몇 개의 근을 갖는지 구하시오.

문항 17 (수정): 루셰의 정리

함수 \(p(z) = z^5 + 3z^2 + 1\)이 고리 영역(annulus) \(A = \{z: 1 < |z| < 2\}\)에 몇 개의 근(zero)을 갖는지 구하시오.

정답 및 해설

루셰의 정리(Rouché's Theorem)를 두 번 적용한다.
1. 원 \(|z|=2\)의 내부:
\(f(z) = z^5\), \(g(z) = 3z^2+1\)로 놓자. 경계 \(|z|=2\)에서:
\(|f(z)| = |z^5| = 2^5 = 32\).
\(|g(z)| = |3z^2+1| \le 3|z|^2+1 = 3(2^2)+1 = 13\).
경계에서 \(|f(z)| > |g(z)|\)이므로, \(p(z) = f(z)+g(z)\)는 \(|z|<2\) 내부에서 \(f(z)=z^5\)와 같은 수의 근을 갖는다. 즉, 5개의 근을 갖는다.

2. 원 \(|z|=1\)의 내부:
\(f(z) = 3z^2\), \(g(z) = z^5+1\)로 놓자. 경계 \(|z|=1\)에서:
\(|f(z)| = |3z^2| = 3|z|^2 = 3\).
\(|g(z)| = |z^5+1| \le |z|^5+1 = 1+1 = 2\).
경계에서 \(|f(z)| > |g(z)|\)이므로, \(p(z)\)는 \(|z|<1\) 내부에서 \(f(z)=3z^2\)와 같은 수의 근을 갖는다. 즉, 2개의 근을 갖는다.

따라서 고리 영역 \(1 < |z| < 2\)에서의 근의 개수는 (\(|z|<2\)에서의 근 개수) - (\(|z| \le 1\)에서의 근 개수) 이다.
경계 \(|z|=1\)에서는 \(|p(z)|=|3z^2+1+z^5| \ge |3z^2| - |z^5+1| \ge 3 - 2 = 1 \neq 0\)이므로 근이 없다.
근의 개수 = \(5 - 2 = 3\)개

문항 18: 선형분수변환

세 점 \(z_1=0, z_2=1, z_3=\infty\)를 각각 \(w_1=-1, w_2=i, w_3=1\)로 보내는 선형분수변환(Möbius transformation) \(w=T(z)\)를 구하시오.

정답 및 해설

선형분수변환의 불변량인 복비(cross-ratio)를 이용한다.
\( \frac{(w-w_1)(w_2-w_3)}{(w-w_3)(w_2-w_1)} = \frac{(z-z_1)(z_2-z_3)}{(z-z_3)(z_2-z_1)} \)
무한대가 포함된 경우, 해당 항을 1로 처리한다.
\( \frac{(w-(-1))(i-1)}{(w-1)(i-(-1))} = \frac{(z-0)(1)}{(1)(1-0)} \)
\( \frac{(w+1)(i-1)}{(w-1)(i+1)} = z \)
\( (w+1)(i-1) = z(w-1)(i+1) \)
\( w(i-1) + (i-1) = zw(i+1) - z(i+1) \)
\( w(i-1) - zw(i+1) = -(i-1) - z(i+1) \)
\( w[(i-1) - z(i+1)] = -i+1 -zi-z \)
\( w = \frac{-i+1 -z(i+1)}{i-1-z(i+1)} = \frac{1-i - (1+i)z}{-1+i - (1+i)z} = \frac{(1-i)-(1+i)z}{(i-1)-(1+i)z}\)
분모 분자에 \(-1\)을 곱하면 \( w = \frac{(1+i)z - (1-i)}{(1+i)z + (1-i)} \).
\( T(z) = \frac{(1+i)z - (1-i)}{(1+i)z + (1-i)} \)

문항 19: 로그함수의 주값

복소수 \(z = (1-i)^{1+i}\)의 주값(principal value)을 구하시오.

정답 및 해설

정의에 의해 \(z^w = e^{w \text{Log}(z)}\)이다. 주값은 대문자 Log를 사용한다.
먼저 \(1-i\)의 극형식을 구한다: \(1-i = \sqrt{2} e^{-i\pi/4}\).
\(\text{Log}(1-i) = \ln|1-i| + i \text{Arg}(1-i) = \ln(\sqrt{2}) - i\frac{\pi}{4} = \frac{1}{2}\ln 2 - i\frac{\pi}{4}\).
따라서, \((1-i)^{1+i} = e^{(1+i)(\frac{1}{2}\ln 2 - i\frac{\pi}{4})}\).
지수를 계산하면: \((1+i)(\frac{1}{2}\ln 2 - i\frac{\pi}{4}) = (\frac{1}{2}\ln 2 + \frac{\pi}{4}) + i(\frac{1}{2}\ln 2 - \frac{\pi}{4})\).
\(z = e^{(\frac{1}{2}\ln 2 + \frac{\pi}{4})} \cdot e^{i(\frac{1}{2}\ln 2 - \frac{\pi}{4})}\).
\(z = e^{\frac{\pi}{4}}\sqrt{2} \left[ \cos(\frac{\ln 2}{2} - \frac{\pi}{4}) + i\sin(\frac{\ln 2}{2} - \frac{\pi}{4}) \right]\).
\(e^{\pi/4}\sqrt{2} \left( \cos(\frac{\ln 2}{2} - \frac{\pi}{4}) + i\sin(\frac{\ln 2}{2} - \frac{\pi}{4}) \right)\)

문항 20: 코시 부등식의 다른 적용

함수 \(f(z)\)가 전해석 함수이고, 모든 \(z\)에 대해 \(|f(z)| \le M|z|^k\)를 만족한다 (단, \(M\)은 양의 상수, \(k\)는 양의 정수). \(f(z)\)가 최대 \(k\)차 다항식임을 증명하시오.

정답 및 해설

코시의 도함수 공식/부등식을 \(f^{(n)}(0)\)에 적용한다.
\(|f^{(n)}(0)| \le \frac{n! M_R}{R^n}\), 여기서 \(M_R\)은 원 \(|z|=R\)에서 \(|f(z)|\)의 최댓값이다.
문제 조건에 의해, \(M_R \le M R^k\).
따라서, \(|f^{(n)}(0)| \le \frac{n! M R^k}{R^n} = n! M R^{k-n}\).
만약 \(n > k\)이면, \(k-n\)은 음수이다. 이때 \(R \to \infty\)의 극한을 취하면, \(|f^{(n)}(0)| \le \lim_{R \to \infty} n! M R^{k-n} = 0\).
따라서 \(n > k\)인 모든 \(n\)에 대해 \(f^{(n)}(0) = 0\)이다.
\(f(z)\)의 Maclaurin 급수는 \(\sum_{n=0}^{\infty} \frac{f^{(n)}(0)}{n!} z^n\)인데, \(n>k\)인 항의 계수가 모두 0이므로, 급수는 유한합이 된다.
\(f(z) = \sum_{n=0}^{k} \frac{f^{(n)}(0)}{n!} z^n\). 이는 최대 \(k\)차 다항식이다.

문항 21: 특이점 종류 판별 II

함수 \(f(z) = z^2 \sin(\frac{1}{z})\)의 특이점 \(z=0\)의 종류를 판별하고, 그 점에서 유수를 구하시오.

정답 및 해설

\(z=0\) 근방에서 로랑 급수를 구한다. \(\sin w = w - \frac{w^3}{3!} + \frac{w^5}{5!} - \dots\)
\(w = 1/z\)를 대입하면,
\(f(z) = z^2 \left( \frac{1}{z} - \frac{1}{3!z^3} + \frac{1}{5!z^5} - \dots \right)\)
\( = z - \frac{1}{6z} + \frac{1}{120z^3} - \dots\)
로랑 급수에서 음수 거듭제곱 항이 무한히 많이 나타나므로, \(z=0\)은 본질적 특이점(essential singularity)이다.
유수는 \(1/z\) 항의 계수이다.
종류: 본질적 특이점, 유수: \(-1/6\)

문항 22: 유수 정리의 응용

경로 C가 원 \( |z|=4 \) (양의 방향)일 때, 다음 적분 값을 구하시오. $$ \oint_C z^2 e^{1/z} \,dz $$

정답 및 해설

피적분함수 \(f(z) = z^2 e^{1/z}\)의 유일한 특이점은 \(z=0\)이며, 이 점은 경로 내부에 있다.
유수 정리에 의해 적분 값은 \(2\pi i \cdot Res(f,0)\)이다.
\(z=0\)에서 유수를 구하기 위해 로랑 급수를 전개한다.
\(e^w = 1 + w + \frac{w^2}{2!} + \frac{w^3}{3!} + \dots\)
\(w=1/z\)를 대입하면, \(e^{1/z} = 1 + \frac{1}{z} + \frac{1}{2!z^2} + \frac{1}{3!z^3} + \dots\)
\(f(z) = z^2 \left( 1 + \frac{1}{z} + \frac{1}{2z^2} + \frac{1}{6z^3} + \dots \right)\)
\( = z^2 + z + \frac{1}{2} + \frac{1}{6z} + \dots\)
\(1/z\) 항의 계수, 즉 유수는 \(1/6\)이다.
적분 값 = \(2\pi i \cdot (1/6) = \pi i / 3\).
정답: \(\pi i / 3\)

문항 23: 조화함수와 평균값 정리

함수 \(u(x,y)=x^2-y^2\)는 영역 \(D=\{(x,y):x^2+y^2 < 4\}\)에서 조화함수이다. 원점 (0,0)에서의 값 \(u(0,0)\)이 반지름 1인 원 \(C: x^2+y^2=1\) 위에서의 \(u\) 값들의 평균과 같음을 보이시오.

정답 및 해설

조화함수의 평균값 성질에 따르면, 조화함수의 한 점에서의 값은 그 점을 중심으로 하는 임의의 원 위에서의 함수값들의 평균과 같다.
이를 직접 계산하여 확인한다. \(u(0,0)=0^2-0^2=0\).
원 C 위의 점은 \(x=\cos\theta, y=\sin\theta\) (\(0 \le \theta \le 2\pi\))로 매개화할 수 있다.
원 C 위에서의 평균값은 \( \frac{1}{2\pi} \int_0^{2\pi} u(\cos\theta, \sin\theta) d\theta \).
\(u(\cos\theta, \sin\theta) = \cos^2\theta - \sin^2\theta = \cos(2\theta)\).
평균값 = \( \frac{1}{2\pi} \int_0^{2\pi} \cos(2\theta) d\theta = \frac{1}{2\pi} [\frac{1}{2}\sin(2\theta)]_0^{2\pi} = \frac{1}{4\pi}(0-0) = 0\).
따라서 \(u(0,0)\) = 0 = 평균값. 성립한다.

문항 24: 코시-리만 방정식의 극형식

함수 \(f(z)=\ln r + i\theta\) (단, \(z=re^{i\theta}, r>0, -\pi < \theta < \pi\))가 해석적인지 판별하시오.

정답 및 해설

\(u(r,\theta) = \ln r\), \(v(r,\theta) = \theta\).
코시-리만 방정식의 극형식은 \(u_r = \frac{1}{r}v_\theta\)와 \(v_r = -\frac{1}{r}u_\theta\)이다.
편도함수를 구하면:
\(u_r = 1/r\), \(u_\theta = 0\)
\(v_r = 0\), \(v_\theta = 1\)
방정식 확인:
\(u_r = 1/r\), \(\frac{1}{r}v_\theta = \frac{1}{r}(1) = 1/r\). (성립)
\(v_r = 0\), \(-\frac{1}{r}u_\theta = -\frac{1}{r}(0) = 0\). (성립)
편도함수들이 연속이고 C-R 방정식을 만족하므로 주어진 영역에서 함수는 해석적이다. (이 함수는 Log z 이다.)
결론: 주어진 영역에서 해석적이다.

문항 25: 함수 방정식

전해석 함수 \(f(z)\)가 모든 \(z\)에 대해 \(f(z+1)=f(z)\)와 \(f(z+i)=f(z)\)를 만족한다. \(f(z)\)는 상수함수임을 증명하시오.

정답 및 해설

주어진 조건은 \(f\)가 주기 1과 주기 i를 갖는 이중주기 함수임을 의미한다.
이는 임의의 정수 m, n에 대해 \(f(z+m+ni) = f(z)\)임을 뜻한다.
단위 정사각형 \(S = \{x+iy | 0 \le x \le 1, 0 \le y \le 1\}\)을 생각하자.
\(f\)는 전해석 함수이므로 연속이고, 닫힌 유계집합 S에서 \(|f(z)|\)는 최댓값 M을 갖는다. (최대-최소 정리)
이중주기성 때문에, 복소평면 전체에서 \(|f(z)|\)의 값은 S에서의 값들과 같다. 즉, 모든 z에 대해 \(|f(z)| \le M\)이다.
따라서 \(f(z)\)는 유계인 전해석 함수이다.
리우빌의 정리에 의해, 유계인 전해석 함수는 상수함수이다.

문항 26: 이상적분의 코시 주값

유수 정리를 이용하여 다음 이상적분의 코시 주값(Cauchy Principal Value)을 구하시오. $$ P.V. \int_{-\infty}^{\infty} \frac{x \sin(x)}{x^2+4} \,dx $$

정답 및 해설

적분은 \(\text{Im} \int_{-\infty}^{\infty} \frac{x e^{ix}}{x^2+4} dx\)와 같다.
복소함수 \(f(z) = \frac{z e^{iz}}{z^2+4}\)를 고려하자. 특이점은 \(z=\pm 2i\).
상반평면에 있는 특이점은 \(z_0 = 2i\) (단순극)이다.
반지름 R인 상반원 경로를 따라 적분하면, 조르단의 보조정리에 의해 \(R \to \infty\)일 때 원호 부분의 적분은 0으로 수렴한다.
따라서, \(\int_{-\infty}^{\infty} f(x)dx = 2\pi i \cdot Res(f, 2i)\).
\(Res(f, 2i) = \lim_{z \to 2i} (z-2i) \frac{z e^{iz}}{(z-2i)(z+2i)} = \frac{2i e^{i(2i)}}{2i+2i} = \frac{2i e^{-2}}{4i} = \frac{e^{-2}}{2}\).
\(\int_{-\infty}^{\infty} \frac{x e^{ix}}{x^2+4} dx = 2\pi i \cdot \frac{e^{-2}}{2} = \pi i e^{-2}\).
우리가 구하는 값은 이 결과의 허수부분이다.
정답: \(\pi e^{-2}\)

문항 27: 항등 정리

단위 원판 \(D=\{z:|z|<1\}\)에서 해석적인 함수 \(f(z)\)가 모든 자연수 \(n \ge 2\)에 대해 \(f(1/n) = 1/n^2\)을 만족한다. \(f(z)\)를 구하시오.

정답 및 해설

함수 \(g(z) = z^2\)를 생각하자. \(g(z)\) 역시 D에서 해석적이다.
수열 \(z_n = 1/n\) (\(n \ge 2\))에 대해, \(f(z_n) = 1/n^2 = (1/n)^2 = g(z_n)\)이다.
수열 \(\{z_n\}\)은 D 내에 있고, 극한점 \(0\)도 D 내에 존재한다. (\(z_n \to 0\) as \(n \to \infty\))
항등 정리(Identity Theorem)에 따르면, 영역 D에서 해석적인 두 함수가 D의 내점을 극한점으로 갖는 수열 위에서 같은 값을 가지면, 두 함수는 D 전체에서 동일하다.
따라서 D에서 \(f(z) = g(z) = z^2\)이다.
정답: \(f(z) = z^2\)

문항 28: 개방 사상 정리

상수함수가 아닌 전해석 함수 \(f(z)\)에 대하여, 상(image) \(f(\mathbb{C})\)가 실수의 부분집합이 될 수 없음을 설명하시오.

정답 및 해설

개방 사상 정리(Open Mapping Theorem)에 따르면, 상수함수가 아닌 해석함수는 열린 집합을 열린 집합으로 보낸다.
복소평면 \(\mathbb{C}\)는 열린 집합이다.
따라서 상 \(f(\mathbb{C})\)는 복소평면 \(\mathbb{C}\)의 열린 부분집합이어야 한다.
하지만, 실수의 부분집합(예: 실수축 \(\mathbb{R}\))은 복소평면에서 열린 집합이 아니다. 실수축 위의 어떤 점을 중심으로 하는 작은 원판도 허수 부분을 가진 복소수를 포함하므로 실수축에 포함될 수 없다.
따라서 \(f(\mathbb{C})\)는 열린 집합이 아니게 되어 모순이 발생한다.
그러므로, 상 \(f(\mathbb{C})\)는 실수의 부분집합이 될 수 없다.

문항 29: 도함수가 0인 경우

영역 D에서 해석적인 함수 \(f(z)=u+iv\)에 대해, 만약 \(u(x,y)\)가 D에서 상수함수이면 \(f(z)\)도 상수함수임을 증명하시오.

정답 및 해설

\(u(x,y)=c\) (상수)라고 가정하자.
그러면 \(u\)의 모든 편도함수는 0이다: \(u_x = 0\), \(u_y = 0\).
\(f\)가 해석적이므로 코시-리만 방정식을 만족한다.
\(v_y = u_x = 0\)
\(v_x = -u_y = 0\)
\(v_y=0\)이므로 \(v\)는 \(y\)에 무관하다, 즉 \(v(x,y)=h(x)\) 형태이다.
\(v_x=0\)이므로 \(h'(x)=0\), 따라서 \(h(x)\)는 상수 \(d\)이다. 즉 \(v(x,y)=d\).
결론적으로 \(f(z) = u+iv = c+id\) 이므로 \(f(z)\)는 상수함수이다.

문항 30: 특이점과 로랑 급수 II

함수 \(f(z) = \frac{z}{e^z-1}\)의 특이점 \(z=0\)의 종류를 판별하고, 이 점 근방에서의 로랑 급수 전개의 처음 세 항을 구하시오.

정답 및 해설

1. 특이점 종류 판별:
\( \lim_{z \to 0} f(z) = \lim_{z \to 0} \frac{z}{e^z-1} \). 이는 0/0 꼴이므로 로피탈의 정리를 사용한다.
\( \lim_{z \to 0} \frac{(z)'}{(e^z-1)'} = \lim_{z \to 0} \frac{1}{e^z} = 1 \).
극한값이 유한하게 존재하므로 \(z=0\)은 제거 가능 특이점이다.
2. 로랑 급수 (이 경우는 테일러 급수) 전개:
\(e^z-1 = (1+z+\frac{z^2}{2!}+\dots)-1 = z+\frac{z^2}{2}+\frac{z^3}{6}+\dots = z(1+\frac{z}{2}+\frac{z^2}{6}+\dots)\).
\(f(z) = \frac{z}{z(1+\frac{z}{2}+\frac{z^2}{6}+\dots)} = \frac{1}{1+(\frac{z}{2}+\frac{z^2}{6}+\dots)}\).
\(w = \frac{z}{2}+\frac{z^2}{6}+\dots\)로 두고, \(\frac{1}{1+w} = 1-w+w^2-\dots\)를 이용한다.
\(f(z) = 1 - (\frac{z}{2}+\frac{z^2}{6}) + (\frac{z}{2})^2 - \dots\)
\( = 1 - \frac{z}{2} - \frac{z^2}{6} + \frac{z^2}{4} - \dots\)
\( = 1 - \frac{z}{2} + (\frac{1}{4}-\frac{1}{6})z^2 - \dots = 1 - \frac{z}{2} + \frac{1}{12}z^2 - \dots \)
(이 계수들은 베르누이 수와 관련있다.)
종류: 제거 가능 특이점, 처음 세 항: \(1 - \frac{1}{2}z + \frac{1}{12}z^2\)
